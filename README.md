# 🎮 数独大师 - 微信小程序

一个精美的数独游戏微信小程序，包含完整的数独算法、三种难度模式和智能提示功能。

## ✨ 功能特性

### 🎯 核心功能
- **智能数独算法** - 自动生成有效数独题目，确保唯一解
- **三种难度** - 简单(30个空)、中等(40个空)、困难(50个空)
- **提示系统** - 每局3次提示机会，帮你突破难关
- **计时功能** - 精确记录通关用时，挑战最快速度
- **实时验证** - 自动检查数字有效性，及时发现错误

### 🎨 界面设计
- **渐变背景** - 紫色系渐变，视觉效果出色
- **流畅动画** - 数字输入、选中高亮都有流畅动画
- **智能高亮** - 选中格子时，行、列、宫格联动高亮
- **响应式布局** - 适配各种屏幕尺寸

### 🎮 游戏体验
- **输入便捷** - 虚拟数字键盘，支持快速输入
- **橡皮擦功能** - 快速清除填错的数字
- **重新开始** - 一键重新开始当前局
- **通关奖励** - 完成后显示用时和统计

## 🚀 快速开始

### 环境要求
- 微信开发者工具 (最新版本)
- 微信基础库 2.19.0+ (建议)

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/Nokkaya/sudoku-wechat.git
   cd sudoku-wechat
   ```

2. **导入项目**
   - 打开微信开发者工具
   - 点击"+"号添加项目
   - 选择 `sudoku-wechat` 文件夹
   - 点击"导入"

3. **编译运行**
   - 点击微信开发者工具右上角的"编译"按钮
   - 在手机模拟器或真机上预览

## 🎮 使用说明

### 选择难度
1. 打开小程序后，看到三个难度选项
2. 点击选择难度（简单/中等/困难）
3. 点击"开始游戏"进入游戏

### 游戏操作
- **选择格子**：点击任意空格选中
- **输入数字**：点击底部数字键盘输入
- **删除数字**：点击橡皮擦图标或选择格子后输入0
- **获取提示**：点击"💡 提示"按钮填写当前格子
- **重新开始**：点击"🔄 重来"按钮重新开局

### 胜利条件
填满所有空格，且所有数字符合数独规则：
- 每行 1-9 各出现一次
- 每列 1-9 各出现一次
- 每个 3x3 宫格 1-9 各出现一次
